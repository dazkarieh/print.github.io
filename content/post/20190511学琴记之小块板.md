## 练习问题

第一周，W老师指出的问题有：

- [ ] 左手按弦时手指未与直板垂直
- [x] 十六分音符和八分音符节奏上的精准切换；
- [ ] 左手在和弦转换时，需要提前「空中成型」；
- [x] 复习基础乐理，提高识谱能力；
- [x] 左手做到此起彼伏；
- [x] 消除左手一切多余动作；
